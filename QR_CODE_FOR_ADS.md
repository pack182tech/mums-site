# QR Code for Pack 182 Mums Site

## QR Code URL
Use this URL to generate QR codes for printed materials, posters, and ads:

```
https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=https://pack182tech.github.io/mums-site
```

## Different Sizes

- **Small (200x200)**: https://api.qrserver.com/v1/create-qr-code/?size=200x200&data=https://pack182tech.github.io/mums-site
- **Medium (300x300)**: https://api.qrserver.com/v1/create-qr-code/?size=300x300&data=https://pack182tech.github.io/mums-site
- **Large (500x500)**: https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=https://pack182tech.github.io/mums-site
- **Extra Large (800x800)**: https://api.qrserver.com/v1/create-qr-code/?size=800x800&data=https://pack182tech.github.io/mums-site

## How to Use

1. **For Digital Ads**: Copy the URL above and paste as an image source
2. **For Print**: 
   - Open any URL above in your browser
   - Right-click and "Save Image As"
   - Use the saved image in your printed materials

## HTML Code for Ads
```html
<img src="https://api.qrserver.com/v1/create-qr-code/?size=300x300&data=https://pack182tech.github.io/mums-site" alt="Scan to Order Mums">
```
